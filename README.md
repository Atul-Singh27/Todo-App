A very small Todo app built with FastAPI. Migrated from Flask. Uses in-memory storage and simple routes to add, update, and delete todos.



How to run

Install dependencies:

pip install fastapi uvicorn python-multipart

Start the server:

uvicorn main:app --reload

Open in browser:

http://127.0.0.1:8000/

http://127.0.0.1:8000/docs
